# Menutree

## GET /api/v1/menutree/id

Retrieve all the elements from the requested menu tree with ID or UUID.

### Returns:

- Name
- Description
- IsSOTrx
- PredefinedContextVariables
- EntityType
- Action
- AD_Window_ID
- AD_Workflow_ID
- AD_Task_ID
- AD_Process_ID
- AD_Form_ID
- AD_InfoWindow_ID
